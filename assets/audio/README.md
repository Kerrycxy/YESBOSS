# YES, BOSS! Audio Assets

本目录按 `音效提示词库.md` 的命名表生成，暂未接入前端代码。

- `ui/`, `hud/`, `card/`, `gossip/`, `press/`, `boardroom/`, `settlement/`, `death/`, `stinger/`, `bgm/`：主提示词库完整目录表。
- `feedback/promise_break.ogg`：demo 任务卡强调的核心反馈音。
- `media/fax_print.ogg`：demo 任务卡中的兼容别名，内容同 `settlement/fax_print.ogg`。
- `_manifest.json`：时长、大小、声道等自检清单。

后续如果用 ElevenLabs/Suno 重新出最终版，保持这些路径和文件名不变即可无痛替换。
